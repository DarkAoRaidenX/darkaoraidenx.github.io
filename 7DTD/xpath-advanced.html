Contains Overview

Contains is a function that targets multiple of something(s) really specific.

Example #1

Say you wanted to do something for just for
Animals and don't want it to affect Zombies.

Instead of doing 1 by 1 you can do "Contains"
It will target every "Node" name that contains the word "animal",
no matter where the word is.
"animalBear" - "Bearanimal"

Text Case matters "animal" vs "Animal".
<set xpath="/entity_classes/entity_class[contains(@name, 'animal')]/property[@name='TimeStayAfterDeath']/@value">1800</set>

Starts-With Overview



Example #1

Say you want to change the Stack Size of items but only certain ones.
You could use "Starts-With".
Let's target "ammo"
<set xpath="/items/item[starts-with(@name, 'ammo')]/property[@name='Stacknumber']">1800</set>
